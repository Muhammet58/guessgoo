# GuessGoo

Minimal ve zarif bir **sayı tahmin oyunu**.  
HTML, CSS ve JavaScript ile geliştirilmiş hızlı ve keyifli bir mini web oyunudur.
Mobil uyumlu yapısı sayesinde hem web tarayıcılarında hem de mobil cihazlarda çalışır.
Oyuncu, seçilen mod ve zorluk seviyesine göre oluşturulan sayıyı verilen ipuçlarına göre bulmaya çalışır.


## Özellikler

- Rastgele sayı üretimi seçilen zorluk seviyesine göre dinamik olarak oluşturuluyor; kolay seviyede kısa, üst seviyelerde daha uzun ve tahmin etmesi zor bir sayı üretiliyor.
- Oyuncunun modu ve zorluğu seçmesine göre değişen tahmin hakları veya süre sınırlaması bulunuyor; skor modunda belirli sayıda deneme hakkın varken, süre modunda zaman dolmadan doğru sayıyı bulman gerekiyor.
- Her tahmin sonrası sistem “çok düşük / çok yüksek” gibi geri bildirimler sunuyor
- En iyi skor (rekor) kaydı
- Modern arayüz
- Tema değiştirici (Light / Dark)
- Buton hover & geçiş animasyonları
- Mobil uyumlu responsive yapı
